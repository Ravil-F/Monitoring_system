# Simple C Web Application Monitoring System

## Описание

Система мониторинга простого веб-приложения на языке C с автоматическим перезапуском при сбоях.

## Компоненты системы

### 1. Веб-приложение на C (`main.c`)
- Простой HTTP сервер на чистом C
- Возвращает "Hello World" на главной странице
- Имеет health-check endpoint `/health`
- Логирует все запросы в файл
- Поддерживает graceful shutdown

### 2. Скрипт мониторинга (`monitor.sh`)
- Проверяет доступность приложения каждые N секунд
- Логирует результаты проверки
- Автоматически перезапускает приложение при сбоях
- Запускается через cron

### 3. Система установки (`install.sh`)
- Автоматическая компиляция и установка всех компонентов
- Настройка systemd сервиса
- Настройка cron заданий
- Настройка ротации логов

## Требования

- Linux система
- gcc компилятор
- curl для взаимодействия с множеством различных серверов по множеству различных протоколов
- systemd для управления сервисами

## Установка

1. Клонируйте или скачайте файлы системы
2. Запустите скрипт установки:

```bash
chmod +x scripts/*.sh
sudo ./scripts/install.sh install
```

## Остановка

```bash
sudo systemctl stop simple-app.service
```
